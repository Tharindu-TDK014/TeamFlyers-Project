# TeamFlyers-Project
E-Commerce Website

Customer - Previlages
_______________________________________________________________
First Visit to website URL
In the Product page go to Login icon
Then if you access the web first time you should Register first
After Registration You can Login 
A Logged User can add products to cart
Added products can be checkout by user
A Logged User can add products reviews
A Logged User can add a post (Complain/Help)


Admin - Previlages
_______________________________________________________________
First admin should login to the Admin 
A Logged admin can add/delete/update stocks 
A Logged admin can add/delete/update products
A Logged admin can add/edit product offers
A Logged admin can access CRM system
A Logged admin can view customer's complains
A Logged admin can view reports
A Logged admin can export reports as pdf
